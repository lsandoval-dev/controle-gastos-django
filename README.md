# 💰 Controle de Gastos Mensais

Sistema web desenvolvido em **Python + Django** para controle financeiro pessoal, permitindo registrar despesas, acompanhar saldo e visualizar gráficos de gastos.

---

## 🚀 Funcionalidades

- 📌 Cadastro de salário mensal
- 💸 Registro de despesas por categoria
- 📊 Visualização de gastos organizados
- 📉 Cálculo automático de saldo disponível
- 📈 Gráficos interativos com Chart.js

---

## 🧠 Objetivo do projeto

Este projeto foi desenvolvido com foco em estudo prático de **Django**, estruturação de aplicações web e manipulação de dados financeiros.

---

## 🛠️ Tecnologias utilizadas

- Python
- Django
- HTML5
- CSS3
- JavaScript
- Chart.js
- SQLite
