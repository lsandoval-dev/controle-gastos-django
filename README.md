# 💰 Sistema de Controle de Gastos Mensais

Projeto web desenvolvido em **Python + Django** para controle financeiro pessoal, permitindo registrar receitas, despesas e acompanhar o saldo de forma simples e intuitiva.

---

## 🚀 Funcionalidades

- 📌 Cadastro de salário mensal
- 💸 Registro de despesas por categoria
- 📊 Visualização de gastos organizados
- 📉 Cálculo automático de saldo disponível
- 📈 Gráficos interativos com Chart.js
- 🧠 Validação e organização de dados

---

## 🛠️ Tecnologias utilizadas

- Python
- Django
- HTML5
- CSS3
- JavaScript
- Chart.js
- SQLite (banco de dados)
